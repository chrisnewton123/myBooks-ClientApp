# Binary-only release repository

This repository contains release artifacts for myBooks client app.

## Included files
- myBooks.exe - Main executable installer
- SHA256SUMS.txt - Checksum for verification
- zuri-inc-sample-data.json - Sample company data for demo/training
- SAMPLE-DATA-README.md - Documentation for the sample data

## Verification
1. Download myBooks.exe
2. Compute SHA256 and compare with SHA256SUMS.txt

## Sample Data

The `zuri-inc-sample-data.json` file contains a complete sample company for demonstration:

- **Company**: Zuri Inc
- **Period**: January - March 2025
- **Sales**: ~$60k per month (~$180k total)
- **Expenses**: ~$45k per month (~$135k total)
- **Includes**: Customers, vendors, invoices, and chart of accounts

See `SAMPLE-DATA-README.md` for import instructions and detailed information.
